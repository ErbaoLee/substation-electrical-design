## 2.3 主变压器选择

### 2.3.1 主变压器容量及台数确定

**确定原则：**

1. n台主变容量总和 ≥ 最大综合计算负荷
   ```
   n·Sn ≥ Smax (MVA)
   ```

2. 一台主变退出时，剩余主变容量应满足60%~70%负荷及全部I+II类负荷
   ```
   Sn ≈ (0.6~0.7)·Smax / (n-1) (MVA)
   Sn ≈ (SI + SII) / (n-1) (MVA)
   ```

**最大综合计算负荷计算：**
```
Smax = Kt·(ΣPi/cosφ)·(1+α%) (MVA)
```

# 第三章 短路电流计算

## 3.1 计算方法和近似

采用标幺阻抗法：

1. 选取基准功率 Sb = 100MVA，基准电压 Ub = Uav
2. 绘制等值网（忽略线路电容、变压器励磁支路）
3. 进行网络变换，求取转移电抗
4. 按串并联规则求各短路点等值阻抗
5. 由基准电流和等值阻抗计算对称短路电流
6. 进一步计算峰值电流、热效应和短路容量
