# 电气主接线方式详解

## 1. 单母线接线

### 接线图
```
[进线1]---[QS1]---[QF1]---[QS2]====[母线W]====[QS3]---[QF2]---[QS4]---[出线1]
                                     |
                                     └---[QS5]---[QF3]---[QS6]---[出线2]
                                     |
                                     └---[出线3]...
```
- QS：隔离开关
- QF：断路器
- W：母线

### 特点
- 优点：接线简单清晰，设备少，投资省，操作方便
- 缺点：母线故障或检修时全站停电，可靠性差
- 适用：小型变电站，可靠性要求不高，回路数少

### 适用范围
| 电压等级 | 最大回路数 |
|---------|-----------|
| 35kV~63kV | ≤5回 |
| 6kV~10kV | ≤6回 |

---

## 2. 单母线分段接线

### 接线图
```
[进线1]---[母线段I]====[QFD]====[母线段II]---[进线2]
    |                                         |
[出线1~n]                                 [出线n+1~m]
```
- QFD：母联断路器
- 正常时分段断路器闭合，两段母线并列运行
- 故障时分段断路器断开，两段母线独立运行

### 特点
- 优点：
  - 一段母线故障不影响另一段
  - 重要用户可从两段母线分别供电
  - 可靠性比单母线显著提高
- 缺点：
  - 母联检修时影响运行灵活性
  - 增加一台断路器和相关隔离开关
- 适用：中型变电站，有重要负荷

### 适用范围
| 电压等级 | 回路数范围 |
|---------|-----------|
| 35kV~63kV | 4~8回 |
| 6kV~10kV | ≥6回 |

---

## 3. 桥型接线

### 3.1 内桥接线

### 接线图
```
[线路1]---[QS1]---[QF1]---[QS2]---[变压器T1]
                      |
[线路2]---[QS3]---[QF2]---[QS4]---[变压器T2]
                      |
                   [QF3](桥断路器)
```
- 桥断路器QF3在变压器侧（内侧）

### 特点
- 优点：设备少（3台断路器），接线简单，经济性好
- 缺点：
  - 桥断路器QF3检修时，两回路解列运行
  - 线路故障时对应变压器停运
- 适用：
  - 35~220kV，两回进出线，两台主变
  - 线路较长，故障率较高
  - 变压器不需要频繁投切

### 3.2 外桥接线

### 接线图
```
[线路1]---[QF1]---[QS1]---[变压器T1]
              |
[线路2]---[QF2]---[QS2]---[变压器T2]
              |
           [QF3](桥断路器)
```
- 桥断路器QF3在线路侧（外侧）

### 特点
- 优点：经济性好，接线简单，线路投切方便
- 缺点：
  - 线路故障或检修时，对应主变需停运
  - 桥断路器QF3检修时回路解列
- 适用：
  - 35~220kV，两回进出线，两台主变
  - 线路短，故障少
  - **有穿越功率时优先选用**
  - 变压器需要频繁投切

### 内桥与外桥比较

| 项目 | 内桥接线 | 外桥接线 |
|------|---------|---------|
| 桥断路器位置 | 变压器侧 | 线路侧 |
| 线路故障影响 | 仅停运故障线路 | 停运对应主变 |
| 变压器故障影响 | 仅停运故障主变 | 停运对应线路 |
| 穿越功率 | 不方便 | 方便 |
| 适用场合 | 线路长、故障多 | 线路短、有穿越功率 |

## 4. 双母线接线

### 接线图
```
            [母线W1]==============================
            /   |   |   |   |   |   |   |   |    \
          QS1  QS2 QS3 QS4 QS5 QS6 QS7 QS8 QS9   QS10
          |    |   |   |   |   |   |   |   |     |
          QF1  QF2 QF3 QF4 QF5 QF6 QF7 QF8 QF9  QFC(母联)
          |    |   |   |   |   |   |   |   |     |
          QS11 QS12QS13QS14QS15QS16QS17QS18QS19 QS20
            \   |   |   |   |   |   |   |   |    /
            [母线W2]==============================
            |    |   |   |   |   |   |   |   |
          [进线1][进线2][出线1][出线2]...[出线n]
```

### 特点
- 优点：
  - 检修任一母线时不中断供电（倒母线操作）
  - 调度灵活，回路可在两组母线间切换
  - 扩建方便，可不停电扩建母线
  - 可靠性高
- 缺点：
  - 设备多（每回路2台断路器+4台隔离开关）
  - 投资大
  - 配电装置复杂
  - 隔离开关作为操作电器，易误操作
- 适用：
  - 110kV重要变电站，出线≥4回
  - 220kV变电站
  - 电源较大，负荷较大

---

## 5. 双母线带旁路母线接线

### 接线图
```
            [母线W1]==============================
            /   |   |   |   |   |   |   |    \
          [双母线接线] + [旁路母线WP] + [旁路断路器QFP]
```

### 特点
- 优点：检修任一台断路器时该回路可不停电
- 缺点：
  - 增加旁路母线和旁路断路器
  - 投资更大
  - 操作复杂
- 适用：
  - 110kV~220kV，回路数较多
  - 断路器检修频繁
  - 对供电连续性要求极高

---

## 6. 3/2 断路器接线

### 接线图
```
[回路1]---[QF1]---[QF2]---[QF3]---[回路2]   (一串)
                   |
[回路3]---[QF4]---[QF5]---[QF6]---[回路4]   (二串)
```
- 每串3台断路器供2个回路
- 正常运行时所有断路器闭合

### 特点
- 优点：
  - **可靠性最高**：任一台断路器检修或故障不影响供电
  - 调度灵活，操作方便
  - 隔离开关仅作检修隔离用，不作为操作电器
  - 运行方式灵活
- 缺点：
  - 设备最多（每回路1.5台断路器）
  - 投资最大
  - 占地面积大
  - 继电保护配置复杂（需识别故障在哪一串内）
- 适用：
  - 220kV及以上特别重要的枢纽变电站
  - 500kV变电站推荐接线
  - 进出线回路数多
  - 对供电连续性要求极高

## 7. 接线方式选择原则

### 7.1 按可靠性要求选择

| 可靠性等级 | 接线方式 | 断路器检修 | 母线检修 | 故障影响范围 |
|-----------|---------|-----------|---------|-------------|
| 低 | 单母线 | 影响本回路 | 全站停电 | 大 |
| 中 | 单母线分段 | 影响本回路 | 一段母线停电 | 中 |
| 中高 | 桥型接线 | 桥断路器检修解列 | 无母线 | 中 |
| 高 | 双母线 | 不影响 | 可倒母线 | 小 |
| 最高 | 3/2接线 | 不影响 | 不影响 | 最小 |

### 7.2 按回路数选择

| 回路数 | 推荐接线 |
|--------|---------|
| 2回 | 桥型接线（内桥/外桥） |
| 3~4回 | 单母线分段/双母线 |
| 4~6回 | 双母线/双母线带旁路 |
| >6回 | 双母线带旁路/3/2接线 |

### 7.3 方案比较维度

#### （1）可靠性比较
- 断路器检修时是否影响供电
- 母线检修时是否停电
- 故障时影响范围
- 是否可能全站停电
- 重要负荷是否得到保证

#### （2）灵活性比较
- 调度是否灵活（回路切换、功率分配）
- 操作是否方便（倒闸操作复杂度）
- 扩建是否方便（是否需停电扩建）

#### （3）经济性比较
- 断路器数量（主要投资）
- 隔离开关数量
- 母线数量及长度
- 占地面积
- 年运行费用

## 8. 各电压等级推荐接线方式汇总

| 电压等级 | 回路数 | 推荐接线 | 备注 |
|---------|--------|---------|------|
| 10kV | ≤6 | 单母线 | 简单经济 |
| 10kV | >6 | 单母线分段 | 有I/II类负荷 |
| 35kV | ≤4 | 单母线 | 出线不超5回 |
| 35kV | >4 | 单母线分段 | 重要负荷多 |
| 110kV | 2 | 内桥/外桥 | 有穿越功率选外桥 |
| 110kV | 3~4 | 单母线分段 | 一般变电站 |
| 110kV | ≥4 | 双母线 | 重要变电站 |
| 220kV | ≤4 | 双母线 | 一般变电站 |
| 220kV | 4~6 | 双母线带旁路 | 检修频繁 |
| 220kV | >4 | 3/2接线 | 枢纽变电站 |
