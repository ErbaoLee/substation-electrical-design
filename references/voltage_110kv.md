# 110kV 变电站电气主接线设计规范

## 适用范围

本规范适用于 110kV 变电站电气主接线设计。

## 典型接线方式

### 1. 内桥接线

**适用条件：**
- 进出线回路数 = 2
- 变压器台数 = 2
- 线路较长，故障率较高
- 无穿越功率

**特点：**
- 优点：设备少，接线简单，经济性好
- 缺点：桥断路器检修时两回路解列运行，可靠性差

### 2. 外桥接线

**适用条件：**
- 进出线回路数 = 2
- 变压器台数 = 2
- 线路短，故障少
- **含穿越功率时优先选用**

**特点：**
- 优点：经济性好，接线简单
- 缺点：线路投切时主变需停运，检修桥断路器回路解列

### 3. 单母线分段接线

**适用条件：**
- 进出线回路数 3~4
- 可靠性要求：重要

**特点：**
- 优点：可靠灵活
- 缺点：设备较多，投资较大

### 4. 双母线接线

**适用条件：**
- 进出线回路数 ≥ 4
- 可靠性要求：特别重要
- **有交换功率或重要枢纽站**

**特点：**
- 优点：可靠性较高，调度容易，扩建方便
- 缺点：设备多，投资大，配电装置复杂，易误操作

## 主变压器选择

### 容量确定原则

1. n台主变容量总和 ≥ 最大综合计算负荷
   ```
   n·Sn ≥ Smax (MVA)
   ```

2. N-1校验：一台主变退出时，剩余主变容量应满足：
   - 60%~70%最大综合负荷
   - 全部I类+II类负荷

### 典型参数

| 项目 | 典型值 |
|------|--------|
| 型号 | SFSZ7-63000/110 |
| 额定容量 | 63 MVA |
| 高压额定电压 | 110±8×1.25% kV |
| 中压额定电压 | 38.5±2×2.5% kV |
| 低压额定电压 | 11 kV |
| 连接组号 | YN,yn0,d11 |
| 阻抗电压 | 高-中：17%；高-低：10.5%；中-低：6.5% |
| 调压方式 | 有载调压 |
| 冷却方式 | 强迫油循环风冷（>31.5MVA） |

## 设备选型

### 断路器
- **类型**：SF6断路器
- **常用型号**：
  - LW11-110（主变/母线侧）：1600A，31.5kA
  - SW6-110（进线侧）：1250A，15.8kA

### 隔离开关
- **常用型号**：
  - GW4-110（主变/母线侧）
  - GW4-110W（进线侧）

### 电流互感器
- **常用型号**：
  - LCWB6-110 2×500/5（主变侧）
  - LCWB6-110 2×200/5（进线侧）
- **准确级**：0.5级
- **二次负荷**：1.2Ω~2Ω(0.5级)
- **二次负荷校验**：S2n ≥ I2n²·Z2L

### 电压互感器
- **常用型号**：JCC-110
- **接线方式**：串级式
- **准确级**：0.5级，容量1000VA

## 设计要点

1. **设备选型**：110kV通常选用SF6断路器
2. **短路电流水平**：一般按31.5kA~40kA考虑
3. **中性点接地方式**：110kV系统采用直接接地
4. **过电压保护**：按DL/T 620配置避雷器
