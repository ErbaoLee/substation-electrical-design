# 35kV变电站典型设计方案（国家电网公司2006年版）

## 概述

本文件依据《国家电网公司输变电工程典型设计：35kV变电站分册》（2006年版）编制，包含13个典型设计方案的技术参数和使用说明。

## 方案分类

35kV变电站典型设计按两个层次分类：

**第一层次 - 按布置方式：**
- **A型**：户外变电站（5个方案：A-1~A-5）
- **B型**：户内变电站（5个方案：B-1~B-5）
- **D型**：箱式变电站（3个方案：D-1~D-3）

**第二层次 - 按配电装置型式：**
- 户外装配式配电装置
- 户内成套开关柜配电装置

## 主要技术原则

### 短路电流水平

| 电压等级 | 短路电流 |
|---------|---------|
| 35kV | **25kA** |
| 10kV | **16kA 或 25kA** |

### 无功补偿配置

- 规程要求：按主变容量的 **10%~30%** 配置
- 典型设计：按主变容量的 **10%~20%** 配置
- 原则：不引起谐波谐振和电压变动过大的前提下，宜加大分组容量、减少分组组数

### 配电装置尺寸推荐值

| 名称 | 推荐值(mm) |
|------|-----------|
| 35kV柜面至柜面之间 | 3600 |
| 35kV柜后至墙中心线之间 | 1500 |
| 35kV柜侧至墙中心线之间 | 1300 |
| 35kV柜面至墙中心线之间 | 2800 |
| 10kV柜面至柜面之间 | 2500 |
| 10kV柜后至墙中心线之间 | 1300 |
| 10kV柜侧至墙中心线之间 | 1300 |
| 10kV柜面至墙中心线之间 | 2200 |

### 假定环境条件

| 项目 | 参数 |
|------|------|
| 海拔高度 | ≤1000m |
| 环境温度 | -20~+40°C |
| 最热月平均最高温度 | 35°C |
| 覆冰厚度 | 10mm |
| 设计风速 | 30m/s（50年一遇10m高10min平均） |
| 污秽等级 | Ⅲ级 |
| 地震设防烈度 | 7度 |
| 土壤电阻率 | ≤100Ω·m |
| 地基承载力 | fak = 150kPa |

### 电气设备爬电比距要求

| 设备位置 | 爬电比距 |
|---------|---------|
| 户外（Ⅲ级污秽区） | ≥31mm/kV（按系统最高运行电压） |
| 户内 | ≥20mm/kV |

## 使用说明

### 方案选择指南

| 条件 | 推荐方案 |
|------|---------|
| 偏远地区、小规模 | A-1, A-2 |
| 一般地区、中等规模 | A-3, A-4, A-5 |
| 城市/城郊、负荷密度高 | B-1, B-2 |
| 城市中心、环境要求高 | B-3, B-4, B-5 |
| 临时用电、快速部署 | D-1, D-2, D-3 |

### 模块拼接方法

实际工程使用典型设计时：
1. 优先选择完整的典型方案
2. 如总布置不满足要求，可选取基本模块拼接
3. 通过子模块调整建设规模
4. 配电装置模块内部尺寸不宜变动
5. 总布置方案可根据实际情况调整

### 需要补充的内容

使用典型设计时，需结合工程实际情况补充：
- 电力系统要求
- 站址地理、地质情况
- 当地水电交通、公共服务设施情况
- 出线走廊规划
- 供水及防洪排水
