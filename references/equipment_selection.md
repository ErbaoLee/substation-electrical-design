# 设备选型手册

## 1. 导体选择

### 1.1 导体类型选择

| 电压等级 | 布置方式 | 导体类型 | 典型型号 |
|---------|---------|---------|---------|
| 110kV及以上 | 户外 | 软导体(钢芯铝绞线) | LGJ系列 |
| 35kV | 户内 | 硬导体(矩形铝排) | LMY系列 |
| 10kV | 户内 | 硬导体(矩形铝排) | LMY系列 |

### 1.2 导体选择条件

**按长期发热允许电流选择：**
```
I_al > Imax / K
```
其中 K 为温度修正系数：`K = √((θ_al - θ) / (θ_al - θ₀))`

**按经济电流密度选择：**
当 Tmax < 5000h 时，按长期发热允许电流选择。

### 1.3 校验项目

| 校验项目 | 条件 | 公式 |
|---------|------|------|
| 热稳定 | S ≥ S_min | S_min = √(Qk·Kf) / C |
| 动稳定(硬导体) | σ_ph ≤ σ_al | σ_ph = f_ph·L² / (10W) |
| 电晕(110kV软导体) | U_cr > U_max | 见下文 |

**热稳定最小截面：**
- C值：硬铝70°C取87，硬铜70°C取171

**电晕校验：**
110kV及以上软导体需校验电晕，当导体外径大于规定值时可免校验：
- 110kV：LGJ-120/70 及以上免校验
- 220kV：LGJ-300 及以上免校验

**电晕临界电压计算：**
```
Ucr = 84·m1·m2·K·δ^(2/3)·(n·r0/K0)·(1 + 0.301/√(r0·δ))·lg(ajj/rd)
```
其中：
- m1：导线表面粗糙系数，一般取0.9
- m2：天气系数，晴天取1.0，雨天取0.85
- K：电容不均匀系数，一般取0.96
- δ：相对空气密度，δ = 2.895p/(273+t)×10^(-3)
- n：分裂导线根数
- r0：导线半径(cm)
- K0：分裂导线表面最大电场强度
- ajj：导线相间几何均距，ajj = 1.26a
- rd：分裂导线等效半径

**要求：Ucr > Umax（最高工作电压）**

### 1.4 动稳定校验（硬导体）

**相间电动力：**
```
fph = 1.73×10^(-7)·(L/a)·ich²  (N/m)
```
其中：
- L：绝缘子跨距(m)
- a：相间距(m)
- ich：冲击电流(A)

**截面系数：**
```
W = b·h²/6  (m³)
```
其中：
- b：导体宽度(m)
- h：导体高度(m)

**相间应力：**
```
σph = fph·L²/(10W)  (Pa)
```

**要求：σph ≤ σal（硬铝允许应力70×10^6 Pa）**

**最大允许跨距：**
```
Lmax = √(10·σal·W/fph)  (m)
```

## 2. 断路器选择

| 电压等级 | 类型 | 说明 |

| 110kV及以上 | SF6断路器 | 灭弧能力强，体积小 |

### 2.2 选择条件

| 项目 | 条件 |

| 额定电流 | In ≥ Imax |

## 4. 电流互感器选择

### 4.1 选择条件

| 项目 | 条件 |
|------|------|
| 额定电压 | Un ≥ Uns |
| 额定电流 | In ≥ Imax |
| 准确级 | 满足测量要求(一般0.5级) |
| 二次容量 | S2n ≥ I2n²·Z2L |

### 4.2 二次负荷计算

```
Z2L = ra + rre + rL + rc
```

其中：
- ra：仪表电流线圈电阻
- rre：继电器电流线圈电阻
- rL：连接导线电阻
- rc：接触电阻，一般取0.1Ω

**最小导线截面：**
```
S ≥ ρ·Lc / (Z2n - ra - rre - rc)
```

### 4.3 常用型号

| LCWB6-110 | 110 | 2×200/5 | 2 | 21 | 54 |
| LCWB6-110 | 110 | 2×500/5 | 1.2 | 45 | 115 |
| LB-35 | 35 | 2×1000/5 | 2 | 24 | 61 |
| LB-35 | 35 | 2×600/5 | 2 | 32 | 81 |
| LFSQ-10 | 10 | 1500/5 | 1.6 | 63 | 130 |

## 5. 电压互感器选择

### 5.1 选择条件

| 项目 | 条件 |
|------|------|
| 额定电压 | 一次侧Un ≥ Uns，二次侧100V |
| 准确级 | 满足测量要求(一般0.5级) |
| 二次容量 | S2n ≥ 二次侧总负荷 |

### 5.2 类型选择

| 电压等级 | 类型 | 说明 |
|---------|------|------|
| 10kV | 浇注式 | JDZJ系列，户内 |
| 35kV | 油浸式 | JDJJ系列，户外/户内 |
| 110kV及以上 | 串级式/电容式 | JCC系列(电磁式)或CVT(电容式) |

### 5.3 接线方式

**三相五柱式（Y/Y/△开口三角）：**
- 用于测量线电压、相电压
- 开口三角用于绝缘监察（零序电压）
- 适用于中性点不接地系统

**单相式：**
- 三台单相互感器接成Y/Y
- 适用于110kV及以上系统

### 5.4 常用型号

| 型号 | 电压(kV) | 接线方式 | 0.5级容量(VA) | 3P级容量(VA) |
|------|---------|---------|--------------|-------------|
| JDZJ-10 | 10 | Y/Y/△ | 120 | 200 |
| JDJJ-35 | 35 | Y/Y/△ | 400 | 600 |
| JCC-110 | 110 | 串级式 | 1000 | 2000 |
| CVT-220 | 220 | 电容式 | 400 | 100 |

### 5.5 二次负荷计算

```
S2 = ΣS仪表 + ΣS继电器
```

**常用仪表负荷：**

| 仪表名称 | 每相负荷(VA) |
|---------|-------------|
| 电压表 | 5~10 |
| 功率表 | 5~10 |
| 电能表 | 5~10 |
| 绝缘监察继电器 | 50~100 |

**要求：S2n ≥ S2**

## 7. 爬电比距要求

### 7.1 户外设备

| 污秽等级 | 爬电比距 | 适用地区 |
|---------|---------|---------|
| Ⅰ级 | ≥16mm/kV | 清洁地区 |
| Ⅱ级 | ≥20mm/kV | 一般地区 |
| Ⅲ级 | ≥31mm/kV | 污秽地区 |
| Ⅳ级 | ≥40mm/kV | 重污秽地区 |

**典型设计假定：Ⅲ级污秽区**

### 7.2 户内设备

- 爬电比距 ≥ 20mm/kV

### 7.3 计算示例

**35kV户外设备（Ⅲ级污秽）：**
```
系统最高运行电压：40.5kV
最小爬电距离：40.5 × 31 = 1256mm
```

**10kV户外设备（Ⅲ级污秽）：**
```
系统最高运行电压：12kV
最小爬电距离：12 × 31 = 372mm
```

## 8. 校验表格式

设备校验结果应按以下格式输出：

```
| 计算数据 | 值 | 设备参数 | 值 | 校验情况 |
|---------|-----|---------|-----|---------|
| Uns | xxx kV | Un | xxx kV | Un > Uns ✓ |
| Imax | xxx A | In | xxx A | In > Imax ✓ |
| I'' | xxx kA | Inbr | xxx kA | Inbr > I'' ✓ |
| ich | xxx kA | Incl | xxx kA | Incl > ich ✓ |
| Qk | xxx A²·s | It²·t | xxx A²·s | It²·t > Qk ✓ |
| ich | xxx kA | ies | xxx kA | ies > ich ✓ |
```
