# 220kV 变电站电气主接线设计规范

## 适用范围

本规范适用于 220kV 变电站电气主接线设计。

## 典型接线方式

### 1. 双母线接线

**适用条件：**
- 进出线回路数 ≤ 4
- 可靠性要求：重要
- 最终规模较小

**特点：**
- 优点：供电可靠性高，调度灵活，扩建方便
- 缺点：设备多，投资大，配电装置复杂，易误操作
- 设备数量：每回路需2台断路器+4台隔离开关

### 2. 双母线带旁路母线接线

**适用条件：**
- 进出线回路数 4~6
- 需保证检修断路器时不中断供电
- 重要负荷供电

**特点：**
- 优点：检修任一台断路器时该回路可不停电
- 缺点：增加旁路母线和旁路断路器，投资更大
- 设备数量：每回路需2台断路器+5~6台隔离开关+旁路母线

### 3. 3/2 断路器接线

**适用条件：**
- 进出线回路数 > 4
- 可靠性要求：特别重要
- 最终规模较大的枢纽变电站
- **220kV系统推荐接线方式**

**特点：**
- 优点：
  - 可靠性最高，任一台断路器检修或故障不影响供电
  - 调度灵活，操作方便
  - 隔离开关仅作检修隔离用，不作为操作电器
- 缺点：
  - 设备最多，投资最大
  - 占地面积大
  - 继电保护配置复杂
- 设备数量：每串3台断路器供2个回路

## 主变压器选择

### 容量确定原则

1. n台主变容量总和 ≥ 最大综合计算负荷
   ```
   n·Sn ≥ Smax (MVA)
   ```

2. N-1校验：一台主变退出时，剩余主变容量应满足：
   - 60%~70%最大综合负荷
   - 全部I类+II类负荷

### 典型参数（220kV三绕组变压器）

| 项目 | 典型值 |
|------|--------|
| 型号 | OSFPSZ-120000/220 或 SFPSZ7-120000/220 |
| 额定容量 | 120/120/60 MVA（高/中/低） |
| 高压额定电压 | 220±8×1.25% kV |
| 中压额定电压 | 121 kV |
| 低压额定电压 | 10.5 kV |
| 连接组号 | YN,yn0,d11 |
| 阻抗电压 | 高-中：13~15%；高-低：23~25%；中-低：7~8% |
| 调压方式 | 有载调压 |
| 冷却方式 | 强迫油循环风冷（ODAF） |
| 相数 | 三相或单相（>120MVA时可考虑单相） |

### 相数选择

- **< 300MVA**：优先选用三相变压器
- **≥ 300MVA**：可选用三相或三个单相变压器组
- **运输条件限制**：可选用单相变压器组

## 导体选择

### 导体类型选择

| 布置方式 | 导体类型 | 典型型号 |
|---------|---------|---------|
| 户外(AIS) | 软导体(钢芯铝绞线) | LGJ-400/50, LGJ-630/55, 2×LGJ-400 |
| 户外(GIS) | 封闭式母线 | SF6气体绝缘母线 |
| 户内 | 管型母线 | Φ80~Φ150 铝锰合金管 |

### 导体选择要点

1. **按长期发热允许电流选择**
2. **按经济电流密度选择**
3. **热稳定校验**：Smin = √(Qk·Kf) / C
4. **动稳定校验**：软导体需校验拉力，硬导体需校验应力
5. **电晕校验**：220kV软导体外径需≥LGJ-300或采用分裂导线

## 设计要点

1. **设备选型**：
   - 优先选用GIS（户内或用地紧张时）或AIS（户外用地充足时）
   - 断路器选用SF6断路器，额定短路开断电流40kA~50kA

2. **短路电流水平**：一般按40kA~50kA考虑

3. **中性点接地方式**：220kV系统采用直接接地，部分变压器中性点可不接地

4. **继电保护**：
   - 双重化配置（两套独立的主保护和后备保护）
   - 主保护：光纤纵差保护
   - 后备保护：距离保护、零序电流保护

5. **过电压保护**：
   - 按DL/T 620配置金属氧化物避雷器(MOA)
   - 雷电活动强烈地区加强保护

6. **配电装置**：
   - AIS：中型布置或半高型布置
   - GIS：户内布置或户外布置
